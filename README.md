## ✨ Fonctionnalités

- 🔤 **Alphabet & Phonétique** — 26 lettres + 44 sons
- 📚 **Vocabulaire** — +250 mots de A1 à C1 avec audio
- 📐 **Grammaire** — Les 12 temps anglais
- 💬 **Phrases** — +80 phrases de la vie courante
- 🎭 **Idiomes** — +40 expressions idiomatiques
- 🎯 **Quiz Adaptatif** — Tous niveaux
- 🎧 **Dictée** — Avec mode ralenti 🐢
- 📖 **Lecture & Compréhension**
- 🗣️ **Dialogues Pratiques**
- ⚠️ **Faux Amis** français/anglais

## 📱 Installer l'app

**Android** : Une bannière apparaît automatiquement → Installer

**iOS** : Safari → Partager ⎙ → "Sur l'écran d'accueil"

## 🛠️ Technologies

- HTML5 / CSS3 / JavaScript vanilla
- Web Speech API
- PWA (offline + installable)
